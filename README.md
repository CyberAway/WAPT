# WAPT

## Install
* git https://github.com/CyberAway/WAPT.git
* cd WAPT
* python3 WAPT.py

## Функционал

* ReverseDNS - Ранжировка IP адрессов
* Domain info - Сбор информации о домене с помощью, whois, nslookup, host
* AUTO Fuzzin - Автоматический фаззинг директорий, файлов, параметров, имён, используется ffuf
